# `app-template`

An boilerplate for rapidly building web applications using Next.js and MUI.

## Install

```bash
pnpm i
```

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
